# 📊 KRX 주식 대시보드

한국거래소(KRX) 상장 주식의 시세 정보를 조회할 수 있는 웹 대시보드입니다.

## 🌟 주요 기능

### 📅 날짜 선택
- 원하는 날짜의 주식 데이터 조회
- 빠른 날짜 버튼: 오늘, 어제, 1주일 전, 1개월 전, 1년 전
- 주말/공휴일은 데이터가 없을 수 있음

### 🔢 표시 개수 선택
- 50개, 100개, 150개, 200개, 250개, 300개 중 선택
- 필터 변경 시 즉시 반영 (재조회 불필요)

### 🔃 정렬 기준 선택
| 정렬 옵션 | 설명 |
|----------|------|
| 시가총액 높은순/낮은순 | 시가총액 기준 정렬 (삼성전자 등 대형주 확인) |
| 종가 높은순/낮은순 | 주당 가격 기준 정렬 |
| 등락률 높은순/낮은순 | 당일 상승/하락률 기준 정렬 |
| 거래량 많은순/적은순 | 거래량 기준 정렬 |
| 종목명 (ㄱ-ㅎ/ㅎ-ㄱ) | 알파벳/가나다 순 정렬 |
| 시장순 | KOSPI/KOSDAQ/KONEX 시장별 정렬 |

### 📈 표시 정보
- **순위**: 선택한 정렬 기준에 따른 순위 (1~3위 특별 표시)
- **종목명**: 주식 이름 및 종목 코드
- **시장**: KOSPI, KOSDAQ, KONEX 구분
- **종가**: 당일 마감 가격 (원)
- **전일대비**: 전일 대비 가격 변동
- **등락률**: 전일 대비 변동률 (%)
- **거래량**: 당일 총 거래량
- **시가총액**: 종가 × 상장주식수 (조/억 단위)

## 🔑 API 정보

### 데이터 출처
- **공공데이터포털**: [data.go.kr](https://data.go.kr)
- **API명**: 금융위원회_주식시세정보
- **엔드포인트**: `getStockPriceInfo`

### API 갱신 주기
- 일 1회 (장 마감 후)

## ⚙️ 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **API**: 공공데이터포털 REST API
- **스타일**: 다크 테마, 반응형 디자인

## 📱 반응형 지원

- 데스크톱: 전체 정보 표시
- 모바일: 거래량, 시가총액 컬럼 숨김

## ⚠️ 주의사항

- 본 데이터는 **참고용**이며, 투자 판단의 근거로 사용할 수 없습니다.
- API 호출 제한이 있을 수 있습니다.
- CORS 정책으로 인해 로컬 서버를 통해 실행해야 합니다.
