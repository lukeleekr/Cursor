# PDF 도구

PDF 페이지 추출 및 파일 병합 기능을 제공하는 데스크톱 프로그램입니다.

## 기능

### 📄 페이지 추출
- PDF 파일에서 원하는 페이지 범위만 추출
- 추출된 파일은 `원본파일명_YYYYMMDD_HHMMSS.pdf` 형식으로 저장
- 저장 위치 지정 가능

### 📎 PDF 병합
- 여러 PDF 파일을 하나로 합치기
- 파일 순서 변경 가능 (1번 파일이 결과물의 맨 앞 페이지)
- 드래그 앤 드롭으로 파일 추가 지원

## 사용 방법

### 실행
`PDF_도구_v4.exe` 파일을 더블클릭하여 실행합니다.

### 페이지 추출
1. **📄 페이지 추출** 탭 선택
2. PDF 파일 선택 (드래그 앤 드롭 또는 찾아보기 버튼)
3. 추출할 페이지 범위 입력 (시작 ~ 끝)
4. 저장 위치 선택 (선택하지 않으면 원본 파일과 같은 위치)
5. **추출하기** 버튼 클릭

### PDF 병합
1. **📎 PDF 병합** 탭 선택
2. 병합할 PDF 파일 추가 (드래그 앤 드롭 또는 추가 버튼)
3. 필요시 순서 변경 (⬆ 위로 / ⬇ 아래 버튼)
4. 저장 위치 및 파일명 선택
5. **병합하기** 버튼 클릭

> 💡 **병합 순서**: 1번 파일 → 2번 파일 → 3번 파일 순서로 병합됩니다.  
> 1번 파일이 결과물의 맨 앞 페이지가 됩니다.

## 개발 환경 설정

### 요구사항
- Python 3.7 이상

### 설치
```bash
pip install -r requirements.txt
```

### 실행 (개발 모드)
```bash
python pdf_extractor_gui.py
```

### 빌드
```bash
python -m PyInstaller --onefile --windowed --name "PDF_도구" pdf_extractor_gui.py
```

## 의존성

- PyPDF2 - PDF 파일 처리
- customtkinter - 현대적인 GUI
- tkinterdnd2 - 드래그 앤 드롭 지원
